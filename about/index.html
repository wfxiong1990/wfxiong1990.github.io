关于我

By Weifeng Xiong

发表于 10月 9 2014

个人简介

  • 本人性别男，爱好女，出生于湖南石门的山旮旯里，今年是人生的第二个本命年。毕业于复旦大学微电子学系，从事金融终端软件开发工作。
  • 工作上用.net 开发，平时热爱折腾新鲜事物，从大学开始折腾重装原版XP，重装Ubuntu，iPhone越狱以及cydia各种插件，android 刷机，重度app爱好者，社交网络控，业余玩一玩python和object-c

为什么要写博客

  • 写给未来的我，我也曾经牛逼过
  • 写给我的后人，你爹会的，你会么？（在这里让我做一个跟汪峰在中国好声音里一样高傲的表情）
  • 作为一个技术积累的收纳箱，把平时琐碎的，掌握的技术，有用的没用的，打上tags，放在博客里，作为参考，也作为以后考察自己程序员技能的一个凭证

为什么不选择CSDN或者博客园或者新浪博客

  • 首先作为一名伪码农，新浪博客之流，个人不做评价，不要问我为什么。
  • CSDN or 博客园，可以平时参考参考，特别是英语力不从心的时候，还是回归主题，程序猿，你用google能搜到多少这类网站的信息？大部分还是stackoverflow github google code 之类的。如果优雅地做一个coder，首先得”崇洋媚外“，多用洋货。
  • 该博客搭建在github pages上面，用了hexo（基于nodejs）的静态博客程序，自定义的程度很高，github上有我一见钟情的pacman博客主题，您现在看到的所有文字和图片，都是用markdown写出来的，博客的每一个改动，都需要git（分布式的版本控制工具）上传至github，所有的这些，够折腾，也够有成就感。
  • 引用苹果肾6的宣传语，比逼格更有逼格。
